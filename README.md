# Role-Based Agent Governance

Portable multi-agent governance skill for role-based planning, review, dispatch, and execution.

This repository uses `SKILL.md` as the model-facing entry point and `OVERVIEW.md` as the public-facing summary.

## Quick Start

1. Install to `~/.codex/skills/role-based-agent-governance`
2. Restart Codex
3. Start with `@intake`

## What Is Included

- `SKILL.md` for the skill entry behavior
- `OVERVIEW.md` for the canonical repository summary
- `BEGINNER_GUIDE.md` and `BEGINNER_GUIDE.zh-CN.md` for onboarding
- `references/` for workflow rules, schemas, and routing
- `skills/` for split workflow skill packages
